Agno is a framework and runtime for building agent platforms.
- Build your agent platform using the Agno SDK.
- Run your agent platform using the AgentOS runtime.
- Manage everything using the AgentOS control plane.

Every company needs an agent platform. Intelligent software that was previously impossible is now one prompt away. Agno helps you build and run your agent platform as a private, secure, unified system.

Companies use Agno across a wide range of work, from building AI-native software like product copilots and agentic widgets, to data labeling, document processing, and internal employee assistants.

Your agent platform runs in your cloud. You own your session, memory, and trace data, and use it to automatically improve your agents. Here's how teams are using Agno:

- **Product teams** use Agno to build product copilots and chatbots. Many build their entire product on top of AgentOS, which exposes a REST API with 80+ endpoints.
- **ML teams** use Agno to label text, image, audio, and video data. Agno is natively typesafe and multi-modal, making it a natural fit for data labeling, extraction, and classification.
- **AI teams** generate synthetic data and preference pairs for training and evals. They automate document processing, knowledge organization, and eval generation with Agno.
- **Data science teams** automate data enrichment, segmentation, and training data curation.
- **Data engineering teams** automate data quality audits, failure log analysis, and weekly reports.

AgentOS runs in your cloud, managed through a beautiful UI. Your data never leaves your environment.
