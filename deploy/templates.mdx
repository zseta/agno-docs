Clone a template, run the setup prompt in your coding agent, and deploy to your cloud.

## Starters

Pick where your platform runs. The core is identical across Starters: platform agents, MCP server, coding-agent skills, and evals. You're choosing the deploy layer.

<CardGroup cols={3}>
  <Card title="Railway" icon="train" href="/deploy/templates/railway/deploy">

  </Card>
  <Card title="Docker" icon="docker" href="/deploy/templates/docker/deploy">
    Run locally or self-host anywhere Docker runs.
  </Card>
  <Card title="AWS" icon="aws" href="/deploy/templates/aws/deploy">
    One script deploys to ECS Express Mode with RDS PostgreSQL.
  </Card>
  <Card title="Fly" icon="plane" href="/deploy/templates/fly/deploy">
    One script deploys a single always-on machine to Fly.io.
  </Card>
  <Card title="GCP" icon="google" href="/deploy/templates/gcp/deploy">
    Deploy to Cloud Run with Cloud SQL in one script.
  </Card>
</CardGroup>

| Starter | Best for | You manage | Cost |
|---------|----------|------------|------|
| Railway | Fast cloud deployment, small teams | Just code | ~$20/mo |
| Docker | Local dev, self-hosted production | Everything | Free |
| AWS | Enterprise deployments, full control | The ECS service and RDS instance | ~$100-110/mo |
| Fly | Single always-on machine, low cost | The Fly app and its Postgres | ~$25/mo |
| GCP | Teams already on Google Cloud | Your GCP project and IAM | ~$135-145/mo |

More Starters are on the way: Kubernetes (Helm), Azure Container Apps, Render, and Modal.

### What's included

| Capability | What it does |
|------------|--------------|
| **Self-learning** | Agents improve from feedback and store learnings for later runs. |
| **Context retrieval** | RAG, knowledge graphs, multi-source navigation. Grounded answers. |
| **Production infrastructure** | PostgreSQL, pgvector, webhook endpoints. Deploys like a Starter. |
| **Interfaces** | Slack wired in. Add Telegram or WhatsApp through [AgentOS interfaces](/deploy/interfaces/overview). |
