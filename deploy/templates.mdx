## Starters

Pick where your platform runs. Every Starter ships the same core: flagship agents, MCP server, coding-agent skills, and evals. Only the deploy layer differs.

  <Card title="Docker" icon="docker" href="/deploy/templates/docker/deploy">

| Starter | Best for | You manage | Cost |
|---------|----------|------------|------|
| Railway | Fast cloud deployment, small teams | Just code | ~$20/mo |
| Docker | Local dev, self-hosted production | Everything | Free |
| AWS | Enterprise deployments, full control | Infrastructure-as-code | ~$75-100/mo |

More Starters are on the way: Fly.io, Google Cloud Run, Kubernetes (Helm), Azure Container Apps, Render, and Modal.

### What's included

| Capability | What it does |
|------------|--------------|
| **Self-learning** | Agents improve from feedback and store learnings for later runs. |
| **Context retrieval** | RAG, knowledge graphs, multi-source navigation. Grounded answers. |
| **Production infrastructure** | PostgreSQL, pgvector, webhook endpoints. Deploys like a Starter. |
| **Interfaces** | Slack, Telegram, WhatsApp ready. |
