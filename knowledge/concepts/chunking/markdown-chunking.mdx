title: Markdown Chunking
---

Markdown chunking is a method of splitting Markdown documents into smaller chunks based on structure like headers, paragraphs, and sections. Set `split_on_headings` to split on heading boundaries, or leave it off for size-based chunking with optional overlap.

<Steps>

  <Step title="Create a Python file">
    ```python markdown_chunking.py
    import asyncio
    from agno.agent import Agent
    from agno.knowledge.chunking.markdown import MarkdownChunking
    from agno.knowledge.knowledge import Knowledge
    from agno.knowledge.reader.markdown_reader import MarkdownReader
    from agno.vectordb.pgvector import PgVector

    db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

    knowledge = Knowledge(
        vector_db=PgVector(table_name="recipes_markdown_chunking", db_url=db_url),
    )

    asyncio.run(knowledge.ainsert(
        url="https://raw.githubusercontent.com/agno-agi/agno/main/README.md",
        reader=MarkdownReader(
            name="Markdown Chunking Reader",
            chunking_strategy=MarkdownChunking(),
        ),
    ))
    agent = Agent(
        knowledge=knowledge,
        search_knowledge=True,
    )

    agent.print_response("What is Agno?", markdown=True)
    ```
  </Step>
