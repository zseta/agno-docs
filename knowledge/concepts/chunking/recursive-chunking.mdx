---
title: Recursive Chunking
---

Recursive chunking is a method of splitting documents into smaller chunks by recursively splitting text at natural break points. This is useful when you want to process large documents in smaller, manageable pieces.

<Steps>

  <Step title="Create a Python file">
    ```python recursive_chunking.py
    import asyncio
    from agno.agent import Agent
    from agno.knowledge.chunking.recursive import RecursiveChunking
    from agno.knowledge.knowledge import Knowledge
    from agno.knowledge.reader.pdf_reader import PDFReader
    from agno.vectordb.pgvector import PgVector

    db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

    knowledge = Knowledge(
        vector_db=PgVector(table_name="recipes_recursive_chunking", db_url=db_url),
    )

    asyncio.run(knowledge.ainsert(
        url="https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf",
        reader=PDFReader(
            name="Recursive Chunking Reader",
            chunking_strategy=RecursiveChunking(),
        ),
    ))

    agent = Agent(
        knowledge=knowledge,
        search_knowledge=True,
    )

    agent.print_response("How to make Thai curry?", markdown=True)
    ```
  </Step>
