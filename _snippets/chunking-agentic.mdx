| Parameter | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| `model` | `Union[Model, str]` | `None` | The model to use for chunking. Accepts a `Model` instance or a model ID string. Defaults to `OpenAIChat` with `gpt-5.4-mini` when not set. |
| `max_chunk_size` | `int` | `None` | The maximum size of each chunk. Defaults to 5000 characters when not set. |
| `custom_prompt` | `str` | `None` | Allows personalized instructions to determine chunk breakpoints. |
