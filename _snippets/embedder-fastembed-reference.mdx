### Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `id` | `str` | The model ID to use for embeddings | `"BAAI/bge-small-en-v1.5"` |
| `dimensions` | `Optional[int]` | Output dimensions of the embedding | `384` |
| `fastembed_client` | `Optional[TextEmbedding]` | Pre-configured FastEmbed `TextEmbedding` client | `None` |
| `enable_batch` | `bool` | Has no effect. FastEmbed embeds one text per call | `False` |
| `batch_size` | `int` | Number of texts per batch. Unused by this embedder | `100` |
