| Parameter            | Type  | Default                   | Description                                            |
| -------------------- | ----- | ------------------------- | ------------------------------------------------------ |
| `server_url`         | `str` | `"http://localhost:9621"` | URL of the LightRAG server.                            |
| `api_key`            | `str` | `None`                    | API key for the LightRAG server.                       |
| `auth_header_name`   | `str` | `"X-API-KEY"`             | Name of the authentication header.                     |
| `auth_header_format` | `str` | `"{api_key}"`             | Format string for the authentication header value.     |
| `name`               | `str` | `None`                    | Name of the vector database.                           |
| `description`        | `str` | `None`                    | Description of the vector database.                    |
