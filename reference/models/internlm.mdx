The InternLM model provides access to the InternLM model.

## Parameters

| Parameter    | Type               | Default                        | Description                                                           |
| ------------ | ------------------ | ------------------------------ | --------------------------------------------------------------------- |
| `id`         | `str`              | `"internlm2.5-latest"`         | The id of the InternLM model to use                                  |
| `name`       | `str`              | `"InternLM"`                   | The name of the model                                                 |
| `provider`   | `str`              | `"InternLM"`                   | The provider of the model                                             |
| `api_key`    | `Optional[str]`    | `None`                         | The API key for InternLM (defaults to INTERNLM_API_KEY env var)      |
| `base_url`   | `Optional[str]`    | `"https://internlm-chat.intern-ai.org.cn/puyu/api/v1/chat/completions"` | The base URL for the InternLM API |

`InternLM` extends [OpenAILike](/reference/models/openai-like) and accepts all of its parameters.
