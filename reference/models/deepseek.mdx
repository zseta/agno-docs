The DeepSeek model provides access to DeepSeek's language models.

## Parameters

| Parameter    | Type               | Default                                        | Description                                                           |
| ------------ | ------------------ | ---------------------------------------------- | --------------------------------------------------------------------- |
| `id`         | `str`              | `"deepseek-v4-flash"`                          | The id of the DeepSeek model to use                                  |
| `name`       | `str`              | `"DeepSeek"`                                   | The name of the model                                                 |
| `provider`   | `str`              | `"DeepSeek"`                                   | The provider of the model                                             |
| `api_key`    | `Optional[str]`    | `None`                                         | The API key for DeepSeek (defaults to DEEPSEEK_API_KEY env var)      |
| `base_url`   | `str`              | `"https://api.deepseek.com"`                   | The base URL for the DeepSeek API                                     |
| `use_thinking` | `Optional[bool]` | `None`                                        | Controls thinking mode. `None` uses the model default (on for thinking-capable models). Set `False` for faster responses without thinking |
| `supports_native_structured_outputs` | `bool` | `False`                     | Whether the model supports native structured outputs                  |

`DeepSeek` extends [OpenAILike](/reference/models/openai-like) and accepts all of its parameters.
