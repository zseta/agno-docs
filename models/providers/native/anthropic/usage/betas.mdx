---
title: Beta Features
description: Enable experimental Anthropic beta features in Agno agents.
---

Anthropic's `betas` are experimental features extending the capabilities of Claude models.

They are supported via the `betas` parameter when initializing the `Claude` model:

```python
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(
        # Activate the context management beta
        betas=["context-management-2025-06-27"],
    ),
)
```

## Example

```python
import anthropic
from agno.agent import Agent
from agno.models.anthropic import Claude

# Setup the beta features we want to use
betas = ["context-1m-2025-08-07"]
model = Claude(betas=betas)

# Note: you can see all beta features available in your Anthropic version like this:
all_betas = anthropic.types.AnthropicBetaParam
print("\n=== All available Anthropic beta features ===")
beta_lines = "\n- ".join(str(b) for b in all_betas.__args__[1].__args__)
print(f"- {beta_lines}")
print("=============================================\n")

agent = Agent(model=model, debug_mode=True)

# The beta features are now activated, the model will have access to use them.
agent.print_response("What is the weather in Tokyo?")
```

## Natively Supported Beta Features

Some beta features need additional configuration. The Context Management beta requires you to activate it with the `betas` parameter and configure edits with the `context_management` parameter. For Agent Skills, setting the `skills` parameter is enough. Agno adds the required betas automatically.

You can check how to use each one of these beta features in our docs.

Here are the ones supported natively with Agno:

- [Context Management](/models/providers/native/anthropic/usage/context-management)
- [Code Execution](/models/providers/native/anthropic/usage/code-execution)
- [File Upload](/models/providers/native/anthropic/usage/file-upload)
- [Prompt Caching](/models/providers/native/anthropic/usage/prompt-caching)
- [Agent Skills](/models/providers/native/anthropic/usage/skills)
- [Web Fetch](/models/providers/native/anthropic/usage/web-fetch)
