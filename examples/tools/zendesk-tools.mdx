---
Enable Agno agents to plug in Zendesk tools for searching and retrieving information from your organization's knowledge base and support documentation to resolve user queries accurately.

```python


from agno.agent import Agent
from agno.tools.zendesk import ZendeskTools

# ---------------------------------------------------------------------------
# Create Agent
# ---------------------------------------------------------------------------


agent = Agent(tools=[ZendeskTools()])

# ---------------------------------------------------------------------------
# Run Agent
# ---------------------------------------------------------------------------
if __name__ == "__main__":
    agent.print_response("How do I login?", markdown=True)
```

## Run the Example
```bash
# Clone and setup repo
git clone https://github.com/agno-agi/agno.git
cd agno/cookbook/91_tools

# Create and activate virtual environment
```
