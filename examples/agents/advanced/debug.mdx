title: "Debug"
description: "Enable verbose debug output for every run with debug_mode, or turn it on for a single run."
source: cookbook/02_agents/14_advanced/debug.py
---

```python debug.py
"""
Debug
=============================

Debug.
"""

from agno.agent import Agent
from agno.models.openai import OpenAIResponses

# You can set the debug mode on the agent for all runs to have more verbose output
# ---------------------------------------------------------------------------
# Create Agent
# ---------------------------------------------------------------------------
agent = Agent(
    model=OpenAIResponses(id="gpt-5-mini"),
    debug_mode=True,
)

# ---------------------------------------------------------------------------
# Run Agent
# ---------------------------------------------------------------------------
if __name__ == "__main__":
    agent.print_response(input="Tell me a joke.")

    # You can also set the debug mode on a single run
    agent = Agent(
        model=OpenAIResponses(id="gpt-5-mini"),
    )
    agent.print_response(input="Tell me a joke.", debug_mode=True)
```
