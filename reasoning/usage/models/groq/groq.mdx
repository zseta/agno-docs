title: Groq Qwen3 32B

<Steps>

  <Step title="Add the following code to your Python file">
    ```python groq_reasoning.py
    from agno.agent import Agent
    from agno.models.groq import Groq

    agent = Agent(
        model=Groq(
            id="qwen/qwen3-32b", temperature=0.6, max_tokens=1024, top_p=0.95
        ),
        reasoning=True,
        markdown=True,
    )
    agent.print_response(
        "9.11 and 9.9 -- which is bigger?", stream=True, show_full_reasoning=True
    )
    ```
  </Step>

  <Snippet file="create-venv-step.mdx" />

  <Step title="Install dependencies">
    ```bash
    uv pip install -U agno groq
    ```
  </Step>

  <Step title="Export your Groq API key">

    <CodeGroup>

    ```bash Windows

    ```
    </CodeGroup>
  </Step>
  <Step title="Run Agent">
    ```bash
    python groq_reasoning.py
    ```
  </Step>
