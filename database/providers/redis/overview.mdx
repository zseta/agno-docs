Agno supports using [Redis](https://redis.io/) as a database with the `RedisDb` class.

## Usage

Install the `redis` and `openai` packages:

```shell
uv pip install redis openai
```

### Run Redis

Install [Docker Desktop](https://docs.docker.com/desktop/) and run **Redis** on port **6379** using:

```bash
docker run -d \
  --name my-redis \
  -p 6379:6379 \
  redis
```

```python redis_for_agent.py
from agno.agent import Agent
from agno.db.redis import RedisDb

# Initialize Redis db (use the right db_url for your setup)
db = RedisDb(db_url="redis://localhost:6379")

# Create agent with Redis db
agent = Agent(db=db)
```
